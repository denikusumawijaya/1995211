🧠 Review: Properties of Logarithms 🧠

Before diving into Surds, let's review Logarithms! They have important properties to simplify expressions:

Sebelum masuk ke Surds, kita review dulu Logaritma! Logaritma punya sifat-sifat penting untuk menyederhanakan ekspresi:

  • Product: $\log_b(m \times n) = \log_b m + \log_b n$
  • Quotient: $\log_b\left(\frac{m}{n}\right) = \log_b m - \log_b n$
  • Power: $\log_b(m^p) = p \cdot \log_b m$
  • Change of Base: $\log_b a = \frac{\log_k a}{\log_k b}$

1. Simplify: $\log(a^2) + \log(b) - \log(a)$ 🧩

1. Sederhanakan: $\log(a^2) + \log(b) - \log(a)$

💡 See Solution Steps / Lihat Langkah Penyelesaian
  1. Use power property: $\log(a^2) = 2\log(a)$.
  2. Use quotient property: $2\log(a) - \log(a) = \log(a)$.
  3. Use product property: $\log(a) + \log(b) = \log(ab)$.

2. Expand: $\log\left(\frac{xy}{z^3}\right)$ 🛠️

2. Jabarkan: $\log\left(\frac{xy}{z^3}\right)$

💡 See Solution Steps / Lihat Langkah Penyelesaian
  1. Separate numerator and denominator: $\log(xy) - \log(z^3)$.
  2. Expand product: $\log x + \log y$.
  3. Expand power: $3\log z$.
  4. Result: $\log x + \log y - 3\log z$.

3. Evaluate $\log_8(32)$ using change of base (base 2)! 🎯

3. Evaluasi $\log_8(32)$ menggunakan perubahan basis (basis 2)!

💡 See Solution Steps / Lihat Langkah Penyelesaian
  1. Convert 8 and 32 to base 2: $8 = 2^3$ and $32 = 2^5$.
  2. Use change of base: $\frac{\log_2(2^5)}{\log_2(2^3)}$.
  3. Result is $\frac{5}{3}$.

4. Rewrite $\log_5(10)$ in terms of common logarithms (base 10)! ✍️

4. Tulis ulang $\log_5(10)$ dalam logaritma basis 10!

💡 See Solution Steps / Lihat Langkah Penyelesaian
  1. Use change of base formula: $\log_b a = \frac{\log_k a}{\log_k b}$.
  2. Apply base 10 (log): $\frac{\log 10}{\log 5}$.

5. Simplify $\log(100x) = $ ... 🏆

5. Sederhanakan $\log(100x)$

💡 See Solution Steps / Lihat Langkah Penyelesaian
  1. Break using product property: $\log(100) + \log(x)$.
  2. $100 = 10^2$, so $\log(100) = 2$.
  3. Result: $2 + \log(x)$.

6. If $\log_b(x/y) = 3$ and $\log_b(y) = 2$, what is $\log_b(x)$? 🤔

6. Jika $\log_b(x/y) = 3$ dan $\log_b(y) = 2$, berapakah $\log_b(x)$?

💡 See Solution Steps / Lihat Langkah Penyelesaian
  1. Property: $\log_b(x/y) = \log_b x - \log_b y$.
  2. Substitute: $3 = \log_b x - 2$.
  3. Move terms: $\log_b x = 3 + 2 = 5$.

7. Evaluate $\log_9(27)$! 💡

7. Evaluasi $\log_9(27)$

💡 See Solution Steps / Lihat Langkah Penyelesaian
  1. Change to base 3: $9 = 3^2$ and $27 = 3^3$.
  2. $\log_9(27) = \frac{\log_3(3^3)}{\log_3(3^2)} = \frac{3}{2}$.

8. Evaluate $(\log_3 5) \times (\log_5 81)$! 🚀

8. Hitung nilai dari $(\log_3 5) \times (\log_5 81)$

💡 See Solution Steps / Lihat Langkah Penyelesaian
  1. Use change of base: $\log_5 81 = \frac{\log_3 81}{\log_3 5}$.
  2. Multiply: $(\log_3 5) \times \left(\frac{\log_3 81}{\log_3 5}\right) = \log_3 81$.
  3. $81 = 3^4$, so the result is 4.

9. Solve: $\log_2(x) + \log_2(x-2) = 3$. What is $x$? 🧠

9. Selesaikan: $\log_2(x) + \log_2(x-2) = 3$. Berapakah $x$?

💡 See Solution Steps / Lihat Langkah Penyelesaian
  1. Product property: $\log_2(x(x-2)) = 3$.
  2. Exponential form: $x(x-2) = 2^3 = 8$.
  3. $x^2 - 2x - 8 = 0 \rightarrow (x-4)(x+2) = 0$.
  4. $x=4$ or $x=-2$. Since log must be positive, $x=4$.

10. Match the logarithmic properties! 🧩

10. Cocokkan sifat logaritma berikut!

$\log_b(m \times n)=$
$\log_b(m/n)=$
$\log_b(m^p)=$
💡 See Solution Steps / Lihat Langkah Penyelesaian

  2. Quotient (division) becomes Subtraction (minus).
  3. Power (exponent) becomes Multiplication (times).

🌍 What are Surds? / Apa itu Surds?

In engineering and architecture, precision is everything. If you round $\sqrt{2}$ to $1.414$, your skyscraper could be off by meters! Surds are 'exact' irrational roots left in root form to maintain 100% precision.

Dalam teknik dan arsitektur, presisi adalah segalanya. Jika kamu membulatkan $\sqrt{2}$ menjadi $1.414$, gedung pencakar langitmu bisa meleset beberapa meter! Surds adalah akar irasional 'eksak' yang dibiarkan dalam bentuk akar untuk menjaga presisi 100%.

  • Rational Number: Can be written as a fraction (e.g., $\frac{1}{2}$, $0.3$, $\sqrt{9}=3$).
  • Bilangan Rasional: Bisa ditulis sebagai pecahan.
  • Irrational Surd: Cannot be written as an exact fraction (e.g., $\sqrt{2}$, $\sqrt{3}$, $\pi$).
  • Surd Irasional: Tidak bisa ditulis sebagai pecahan eksak.

1. Which of the following is a Surd? 🧩

1. Manakah yang merupakan Surd?

💡 See Solution Steps / Lihat Langkah
  1. A surd is an irrational root. $\sqrt{4} = 2$ and $\sqrt{16} = 4$ (both rational).
  2. $\sqrt{5}$ gives a non-repeating, non-terminating decimal, so it's an irrational surd.

2. Why is $\sqrt{2}$ more precise than 1.414 in engineering? 🏗️

2. Mengapa $\sqrt{2}$ lebih presisi daripada 1.414 dalam teknik?

💡 See Solution Steps / Lihat Langkah
  1. 1.414 is an approximation. $\sqrt{2}$ is the exact value.
  2. Using approximations in large scales leads to significant errors.

3. Is $\frac{3}{4}$ a rational number or an irrational surd? 🤔

3. Apakah $\frac{3}{4}$ bilangan rasional atau surd irasional?

💡 See Solution Steps / Lihat Langkah
  1. It can be written as a fraction of two integers.
  2. Therefore, it is a rational number.

4. Is $\sqrt{9}$ a surd? 🎯

4. Apakah $\sqrt{9}$ adalah surd?

💡 See Solution Steps / Lihat Langkah
  1. A surd must be irrational.
  2. $\sqrt{9} = 3$, which is an exact integer (rational). So it's not a surd.

5. Which number is irrational? ✍️

5. Nomor manakah yang irasional?

💡 See Solution Steps / Lihat Langkah
  1. $0.5 = \frac{1}{2}$ and $2 = \frac{2}{1}$ (both rational).
  2. $\sqrt{7}$ cannot be expressed as a simple fraction.

6. What is the exact value of $\sqrt{3}$? 💡

6. Berapakah nilai eksak dari $\sqrt{3}$?

💡 See Solution Steps / Lihat Langkah
  1. The exact value is left in root form to maintain precision.
  2. Therefore, the exact value is $\sqrt{3}$.

7. True or False: $0.333...$ is an irrational number. 🧠

7. Benar atau Salah: $0.333...$ adalah bilangan irasional.

  2. Since it can be written as a fraction, it is rational, not irrational.

8. Which of these represents an exact value? 🚀

8. Manakah yang merepresentasikan nilai eksak?

💡 See Solution Steps / Lihat Langkah
  1. $3.14$ and $\frac{22}{7}$ are approximations of $\pi$ or other roots.
  2. $\sqrt{10}$ is the exact mathematical representation.

9. Is $\sqrt{0}$ a surd? 🤨

9. Apakah $\sqrt{0}$ adalah surd?

💡 See Solution Steps / Lihat Langkah
  1. $\sqrt{0} = 0$, which is a rational number.
  2. Surds must be positive irrational roots, so $\sqrt{0}$ is not a surd.

🧩 Perfect Squares: The Keys to Surds! / Bilangan Kuadrat Sempurna: Kunci Surds!

To simplify a surd, we need to find the hidden perfect squares inside them. Perfect squares are numbers like 1, 4, 9, 16... that come from squaring an integer.

Untuk menyederhanakan surd, kita harus menemukan kuadrat sempurna yang tersembunyi di dalamnya. Kuadrat sempurna adalah angka seperti 1, 4, 9, 16... yang berasal dari mengkuadratkan bilangan bulat.

$1^2=1$, $2^2=4$, $3^2=9$, $4^2=16$, $5^2=25$, $6^2=36$, $7^2=49$, $8^2=64$, $9^2=81$, $10^2=100$, $11^2=121$, $12^2=144$

1. What is the largest perfect square factor of 20? 🗝️

1. Apa faktor kuadrat sempurna terbesar dari 20?

💡 See Solution Steps / Lihat Langkah
  1. Factors of 20: 1, 2, 4, 5, 10, 20.
  2. Which are perfect squares? 1 and 4.
  3. The largest is 4.

2. What is $\sqrt{64}$? 🎯

2. Berapakah $\sqrt{64}$?

💡 See Solution Steps / Lihat Langkah
  1. Ask yourself: What number times itself equals 64?
  2. $8 \times 8 = 64$.
  3. So, $\sqrt{64} = 8$.

3. Is 50 a perfect square? 🤔

3. Apakah 50 bilangan kuadrat sempurna?

💡 See Solution Steps / Lihat Langkah
  1. $7^2 = 49$ and $8^2 = 64$.
  2. 50 is between them, so no integer squared equals 50.

4. What is the largest perfect square factor of 32? 🛠️

4. Apa faktor kuadrat sempurna terbesar dari 32?

💡 See Solution Steps / Lihat Langkah
  1. Factors of 32: 1, 2, 4, 8, 16, 32.
  2. Perfect squares among them: 1, 4, 16.
  3. The largest is 16.

5. What is $11^2$? ✍️

5. Berapakah $11^2$?

💡 See Solution Steps / Lihat Langkah

  2. $110 + 11 = 121$.

6. What is the largest perfect square factor of 48? 🏆

6. Apa faktor kuadrat sempurna terbesar dari 48?

💡 See Solution Steps / Lihat Langkah
  1. Factors of 48: 1, 2, 3, 4, 6, 8, 12, 16, 24, 48.

  3. The largest is 16.

7. What is $\sqrt{144}$? 🧠

7. Berapakah $\sqrt{144}$?

🤝 Simplifying & Rationalizing Surds / Menyederhanakan & Merasionalisasi Surds

Let's unlock surds using perfect squares! $\sqrt{50} = \sqrt{25 \times 2} = 5\sqrt{2}$. When adding, treat surds like variables: $3\sqrt{2} + 5\sqrt{2} = 8\sqrt{2}$. But $\sqrt{2} + \sqrt{3}$ cannot be added! What if the surd is in the denominator? We 'rationalize' it by multiplying by 1 in a clever way.

Mari buka surds menggunakan kuadrat sempurna! $\sqrt{50} = \sqrt{25 \times 2} = 5\sqrt{2}$. Saat menambahkan, perlakukan surds seperti variabel: $3\sqrt{2} + 5\sqrt{2} = 8\sqrt{2}$. Tapi $\sqrt{2} + \sqrt{3}$ tidak bisa ditambahkan! Bagaimana jika surd ada di penyebut? Kita 'rasionalisasi' dengan mengalikan 1 dengan cara cerdas.

  • Rationalization: $\frac{1}{\sqrt{3}} \times \frac{\sqrt{3}}{\sqrt{3}} = \frac{\sqrt{3}}{3}$

1. Simplify $\sqrt{50}$! 🗝️

1. Sederhanakan $\sqrt{50}$!

💡 See Solution Steps / Lihat Langkah
  1. Largest perfect square in 50 is 25. $50 = 25 \times 2$.
  2. $\sqrt{25 \times 2} = \sqrt{25} \times \sqrt{2}$.
  3. $5 \times \sqrt{2} = 5\sqrt{2}$.

2. Simplify $\sqrt{72}$! 🛠️

2. Sederhanakan $\sqrt{72}$!

💡 See Solution Steps / Lihat Langkah
  1. Largest perfect square in 72 is 36. $72 = 36 \times 2$.
  2. $\sqrt{36 \times 2} = 6 \times \sqrt{2} = 6\sqrt{2}$.

3. Calculate $3\sqrt{5} + 4\sqrt{5}$! 🎯

3. Hitunglah $3\sqrt{5} + 4\sqrt{5}$!

💡 See Solution Steps / Lihat Langkah
  1. Like surds can be added (just like $3x + 4x = 7x$).
  2. $3\sqrt{5} + 4\sqrt{5} = (3+4)\sqrt{5} = 7\sqrt{5}$.

4. Can $\sqrt{2} + \sqrt{3}$ be simplified to $\sqrt{5}$? 🤔

4. Bisakah $\sqrt{2} + \sqrt{3}$ disederhanakan menjadi $\sqrt{5}$?

💡 See Solution Steps / Lihat Langkah
  1. Unlike surds cannot be added directly.
  2. $\sqrt{2} + \sqrt{3}$ is already in its simplest form. It is NOT $\sqrt{5}$.

5. Simplify $2\sqrt{3} + \sqrt{27}$! 💡

5. Sederhanakan $2\sqrt{3} + \sqrt{27}$!

💡 See Solution Steps / Lihat Langkah
  1. First, simplify $\sqrt{27}$. Largest square is 9. $\sqrt{9 \times 3} = 3\sqrt{3}$.
  2. Now add: $2\sqrt{3} + 3\sqrt{3} = 5\sqrt{3}$.

6. Multiply $(2\sqrt{5})(3\sqrt{2})$! ✍️

6. Kalikan $(2\sqrt{5})(3\sqrt{2})$!

💡 See Solution Steps / Lihat Langkah
  1. Multiply coefficients: $2 \times 3 = 6$.
  2. Multiply surds: $\sqrt{5} \times \sqrt{2} = \sqrt{10}$.
  3. Result: $6\sqrt{10}$.

7. To rationalize $\frac{1}{\sqrt{3}}$, what do we multiply by? 🧠

7. Untuk merasionalisasi $\frac{1}{\sqrt{3}}$, kita kalikan dengan apa?

💡 See Solution Steps / Lihat Langkah
  1. We want the denominator to become a rational number.
  2. Multiply top and bottom by $\sqrt{3}$ so denominator becomes $\sqrt{3} \times \sqrt{3} = 3$.

8. Rationalize $\frac{5}{\sqrt{3}}$! 🚀

8. Rasionalisasikan $\frac{5}{\sqrt{3}}$!

💡 See Solution Steps / Lihat Langkah
  1. Multiply by $\frac{\sqrt{3}}{\sqrt{3}}$.
  2. Numerator: $5 \times \sqrt{3} = 5\sqrt{3}$.
  3. Denominator: $\sqrt{3} \times \sqrt{3} = 3$.
  4. Result: $\frac{5\sqrt{3}}{3}$.

9. Simplify $\sqrt{12} - \sqrt{3}$! 🏆

9. Sederhanakan $\sqrt{12} - \sqrt{3}$!

💡 See Solution Steps / Lihat Langkah
  1. Simplify $\sqrt{12}$. Largest square is 4. $\sqrt{4 \times 3} = 2\sqrt{3}$.
  2. Subtract: $2\sqrt{3} - \sqrt{3} = \sqrt{3}$.

10. Match the surds with their simplified forms! 🧩

10. Cocokkan surds dengan bentuk sederhananya!

$\sqrt{18}$:
$\sqrt{20}$:
$\sqrt{48}$:
💡 See Solution Steps / Lihat Langkah
  1. $\sqrt{18} = \sqrt{9 \times 2} = 3\sqrt{2}$.
  2. $\sqrt{20} = \sqrt{4 \times 5} = 2\sqrt{5}$.
  3. $\sqrt{48} = \sqrt{16 \times 3} = 4\sqrt{3}$.
